Development Diary

The planning stage began by identifying the overall purpose of the website. The main goal was to create a fitness-themed website that helps gym users plan and understand workout routines in a simple and clear way. Many people feel unsure about what exercises to do at the gym, so the website aims to reduce confusion and make training more approachable. The target audience includes teenagers and adults who regularly go to the gym or are just starting out and want structured guidance.

During the planning process, I created a basic site map to decide how users would navigate between pages. This helped me organise the website into clear sections such as Home, Generator, Saved Plans, About, and the Development Diary. Planning the structure early made it easier to build the website logically and ensured that users could move between pages without getting lost.

In the design phase, I chose a dark neon theme inspired by modern fitness apps and gym websites. This style was chosen because dark backgrounds with bright accent colours are popular in the fitness industry and create a strong, energetic feel. I used a dark gradient background combined with purple and pink neon highlights to make headings, buttons, and key elements stand out. Glass-style cards were used to separate content from the background and improve readability.

I also focused on keeping the design consistent across all pages. The same navigation bar, colour scheme, and card layout were used throughout the website so users always feel familiar with the interface. Buttons and hover effects were added to make the site feel interactive and more engaging, even when viewing static content.

During the development stage, I used HTML to create the structure of each page. Semantic elements such as nav, header, section, and footer were used to organise content clearly and improve accessibility. Headings, paragraphs, lists, images, and forms were used where appropriate to present information in a structured way.

CSS was then used to style the website and bring the design to life. Flexbox was used to align content and create responsive layouts that adapt to different screen sizes. CSS was also used to control colours, fonts, spacing, hover effects, and visual hierarchy. This helped ensure the website looks good on both desktop and mobile devices.

Testing was an important part of the development process. I tested the website by opening it on different screen sizes and checking that all pages loaded correctly. I made sure the navigation links worked as expected and that buttons responded properly when hovered over or clicked. I also checked colour contrast to ensure text was readable and accessible for users.

Based on testing, small improvements were made to spacing, text size, and layout to improve usability. This process helped identify minor issues early and ensured the final website feels polished and user-friendly.

If the project were to be developed further, future improvements could include adding more advanced JavaScript features such as saving workout plans automatically and generating routines based on user input. Additional features like BMI calculators, progress tracking, and animated transitions could also be added to improve functionality and user engagement. Overall, the project successfully meets its aims and provides a clear, modern fitness planning website.
